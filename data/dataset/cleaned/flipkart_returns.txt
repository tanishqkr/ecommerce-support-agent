[DOCUMENT: Flipkart Order Cancellation and Return Policy]

[SECTION: Cancellation Eligibility — Standard Orders]
* Order is eligible for cancellation only if it has not yet been dispatched.
* Order is not eligible for cancellation once it is out for delivery.
* Order is eligible for rejection at the doorstep if already out for delivery.
* Cancellation is eligible within the time window specified on the product page or order confirmation page.
* Order is not eligible for free cancellation after the specified time window has passed.
* Cancellation fee shall be charged if the order is cancelled after the specified free-cancellation window.
* Refund is issued in full for prepaid orders when the seller cancels due to unforeseen circumstances.
* Flipkart reserves the right to accept or reject any cancellation request.

[SECTION: Cancellation Eligibility — Hyperlocal (MINUTES Delivery)]
* Orders placed under the MINUTES delivery option are not eligible for self-serve cancellation.
* Orders placed under the MINUTES delivery option are not eligible for self-serve refund.
* Cancellation is eligible only if requested via a CX Agent under the following conditions:
* Cancellation is eligible if the order could not be delivered within the estimated time displayed at order placement.
* Cancellation is eligible if the order has not been picked up by the delivery partner.
* Cancellation is eligible if the seller has not accepted the order for reasons not attributable to the customer.
* Cancellation is eligible if Easy Doorstep Cancellation is applicable.
* Refund is issued within 5 to 7 business days for cancellations initiated by Flipkart.
* Customer is not charged for any order cancelled by Flipkart.

[SECTION: General Returns — Governing Rules]
* Return policy provided on the individual product page shall prevail over the general returns policy.
* All products listed under a category may not have the same return policy.
* Return for change of mind is eligible only if the product category allows remorse returns.
* Flipkart reserves the right to restrict the number of returns per order unit after evaluation of defect.

[SECTION: Non-Returnable Categories]
* Item is not eligible for return if it belongs to a non-returnable category as marked on the product page.

* The following products are not eligible for return due to hygiene, health, personal care, or wellness reasons: diaper, bottle nipple, female urination device, women intimate care, teether soother, nipple puller, sanitary pad, pantyliner, condom, toothbrush, tampon, dental floss stick, toilet brush, cleaning glove, boxer, trunk, sock, panty.

[SECTION: Return Pickup Conditions — Part 2]
* Return pickup address change is eligible only if pickup service is available at the new address.
* Return is eligible only if the product returned is the correct product (IMEI, name, image, brand, serial number, article number, barcode must match and MRP tag must be undetached and clearly visible).
* Return is eligible only if the product is complete (all in-the-box accessories, freebies, and combos must be present).
* Return is eligible only if the product is unused, unwashed, unsoiled, without stains, and with non-tampered quality check seals, return tags, and warranty seals where applicable.
* Return is not eligible if a Mobile, Laptop, or Tablet has not been formatted and Screen Lock (Pin, Pattern, or Fingerprint) has not been disabled before return.
* Return is not eligible if an Apple device has iCloud lock still enabled at the time of pickup.
* Return is not eligible if the product is damaged with scratches, dents, tears, or holes (including SIM trays, charging port, headphone port, back panel).
* Return is not eligible if the product's original packaging or box is damaged.
* Field executive shall refuse to accept the return if any of the above pickup conditions are not met.
* Refund is issued only after the returned product has been received by the seller.

[SECTION: Special Conditions — General Rules for Returns]
* If the seller is unable to process a replacement for any reason, a refund shall be issued instead.
* If a product accessory is found missing, damaged, or defective, the seller shall either process a replacement of the accessory or issue an eGV equivalent to the price of the accessory.
* During open box delivery: If a different or damaged product is received, refund is eligible on the spot (cash-on-delivery orders receive spot refund).
* Return is not eligible after accepting an open box delivery, except for manufacturing defects.
* For products where installation is provided by Flipkart's service partners: Return eligibility requires that the customer must not open the product packaging before Flipkart authorized personnel perform unboxing and installation.
* For Furniture: Full replacement is eligible only if the authorized service personnel determines that replacing the faulty or defective part will not resolve the issue.
* For Furniture: Authorized service personnel shall first attempt to resolve issues by replacing the faulty or defective part at no cost.
